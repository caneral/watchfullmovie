
# Watch Full Movie
It is a movie web application developed with ReactJS.
## Installation



```bash
# If you use npm:
npm install 
# and
npm start

# Or if you use Yarn:
yarn add 
# and
yarn start
```

## What Does It Contain?

- ReactJS
- Redux
- TailwindCSS
- Material-UI
- MovieDB API
